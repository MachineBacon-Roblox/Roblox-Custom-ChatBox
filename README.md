# Introductions
A custom chatbox (commentbox) that don't requires roblox age check for chatting. 
# Setting Up 
Download ALL files in the folder "ChatBox" and import them to roblox studio. Then open the folder, find the file "ChatBoxGui" and put it under "StarterGui". Next, find the folder "ChatBox(Server)" and put it under "ServerScriptService",  folder "ChatBox(ReplicatedStorage)" put it under "ReplicatedStorage". lastly, you can delete the folder "ChatBox". Congratulations, you have finished all the setups. 
# Guides to Use
Command list:
<br/>1.  /clear : clear all chats(comments) in the chatbox(commentbox). 
<br/>2.  /ban : ban or unban the selected player
<br/>3.  /show all : show all private chats(comments). 
<br/>4.  /load history : load all chat history(comment history). 
<br/>5.  /clear history : clear all chat history(comment history).
